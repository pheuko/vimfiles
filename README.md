# Vimfiles

Vim files with Vim bundle

## Install

```
git clone repository ~/.vim
```

Symlink vimrc and gvimrc

Install Vundle

```
BundleInstall
```

## Extras

### Command-t:

```zsh
cd ~/.vim/bundle/Command-T && ruby extconf.rb
```

```zsh
make
```

### YouCompleteMe

```zsh
cd ~/.vim/bundle/YouCompleteMe && ./install.sh
```

#### With TernJS

```zsh
cd ~/.vim/bundle/tern_for_vim && npm install
```
